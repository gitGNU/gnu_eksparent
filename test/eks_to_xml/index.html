The EKS Language

What is Eks?

The Eks Language is a markup language designed to be used as a more type friendly variant of the already existing ones.

What can Eks do?

Eks can do highly complicated hireachies and tree structures. It can also do single-line and multiline comments (/* multiline */ , // singleline). It should in its future also be able to execute code in itself.

How far has Eks reached?

Eks is a work in progress, it isnt standardized yet and is mostly under development. However, many functions is already implemented and there should not be much time left until version 1.0. This web-page is for example written in eks.
